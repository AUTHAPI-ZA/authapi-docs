# Public API Documentation

This documentation provides information about the public API version 2.

import { Callout, Steps } from "nextra-theme-docs";


When you are here, you probably already have an API key. If not, you can get one by registering at [New App](https://apiauth.vercel.app/dashboard/new).

The API key is used to authenticate your requests. You can pass it as a query parameter or as a header.

<Steps>

### Add env variables

```bash
_AUTHAPI_SECRET_=... # A secret key for your app
_AUTHAPI_APPID_=... # An app id for your app
_AUTH_API_URL_=https://weary-puce-cardigan.cyclic.app/
```

### App Authentication

```js copy
const authHeader = {
  headers: {
    Authorization: `Bearer ${process.env._AUTHAPI_APPID_}:${process.env._AUTHAPI_SECRET_}`,
    // Add own headers here
  },
};
```

</Steps>
